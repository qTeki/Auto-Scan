# Auto Scan
 A Bash script for automated port and service scanning using nmap.

  Discailmer: This script is intended to be used in CTF only due to its loudness and unsafe code.
# Usage:

Run the command as root

```./autoscan.sh```

Then privide the IP

```[~] IP: <IP-ADDRESS>```
